# Web桌面系统

一个基于PHP和现代Web技术构建的在线桌面环境，提供完整的桌面体验。

## 功能特性

### 桌面环境
- 🖥️ 完整的桌面界面，包含任务栏、开始菜单、桌面图标
- 🎨 现代化的UI设计，支持主题切换
- 📱 响应式设计，支持移动设备
- ⚡ 流畅的动画效果和交互体验

### 窗口管理
- 📦 完整的窗口系统：最小化、最大化、关闭、拖拽、调整大小
- 🔄 Alt+Tab窗口切换
- 📌 任务栏显示运行中的应用
- 🎯 多窗口管理和Z轴层级控制

### 内置应用程序

#### 🌐 浏览器
- 支持网页浏览和导航
- 前进/后退/刷新功能
- 地址栏和搜索功能
- 书签快速链接

#### 📁 文件管理器
- 完整的文件系统浏览
- 文件和文件夹操作（创建、删除、重命名）
- 多种视图模式
- 文件类型图标显示

#### 🧮 计算器
- 标准计算器功能
- 键盘快捷键支持
- 清晰的数字显示

#### 📝 文本编辑器
- 文件打开/保存/另存为
- 查找和替换功能
- 语法高亮（基础）
- 撤销/重做操作

#### 💻 终端
- 模拟Linux终端环境
- 支持常用命令（ls, cd, pwd, mkdir, touch, rm, cat等）
- 命令历史记录
- 文件系统操作

#### ⚙️ 设置
- 系统设置管理
- 主题和外观配置
- 设置导入/导出
- 系统信息查看

## 技术架构

### 前端技术
- **HTML5/CSS3**: 现代Web标准
- **JavaScript ES6+**: 原生JavaScript，无依赖框架
- **Font Awesome**: 图标字体库
- **CSS Grid/Flexbox**: 响应式布局

### 后端技术
- **PHP 7.4+**: 服务端脚本语言
- **文件系统API**: 文件操作和管理
- **JSON API**: RESTful接口设计

### 架构特点
- 🏗️ 模块化组件设计
- 🔧 面向对象编程
- 📡 前后端分离架构
- 🛡️ 安全的文件访问控制

## 安装和使用

### 环境要求
- PHP 7.4 或更高版本
- Apache/Nginx Web服务器
- 现代浏览器（Chrome、Firefox、Safari、Edge）

### 快速开始

1. **下载项目**
   ```bash
   git clone <repository-url>
   cd testwebos
   ```

2. **配置服务器**
   - 将项目文件部署到Web服务器
   - 确保PHP已启用
   - 配置`.htaccess`（Apache）或相应的服务器配置

3. **访问系统**
   - 在浏览器中访问项目URL
   - 系统将自动加载桌面环境

### 本地开发
```bash
# 使用PHP内置服务器（仅用于开发）
php -S localhost:8000

# 然后在浏览器中访问 http://localhost:8000
```

## 开发指南

### 添加新应用
1. 在`js/apps/`目录创建新的应用文件
2. 继承基础应用类结构
3. 在`js/desktop.js`中注册应用
4. 添加相应的CSS样式

### 自定义主题
1. 修改`css/desktop.css`中的CSS变量
2. 添加新的主题类
3. 在设置应用中添加主题选项

### API扩展
1. 在`php/api.php`中添加新的端点
2. 实现相应的处理方法
3. 更新前端调用代码
